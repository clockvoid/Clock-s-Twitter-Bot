初めての投稿となります、TwitterのBotをJavaで作ってみました。クソースなので、治したいので参考となるような意見をいただければ幸いです。

このBotには各自プラグインを追加することができます。
プラグインは、src/Bot/UpdateBot.javaにある、UpdateBotインターフェースを実装して、
Jarに固め、pluginsフォルダに追加するだけで簡単に追加することができるようになっています。
searchWord()メソッドが最初に呼ばれ、反応したい内容や、変数の初期化などを行います。返却値は、反応したい内容が現れた最初の文字がツイートの何番目にあるかを返します。
update()メソッドでTwitterへの変更を行います。
※プラグインを作るにはtwitter4jの知識と、twitter4jへのクラスパスが必要です。
